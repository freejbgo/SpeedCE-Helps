## 案例 1：CDN 回源 IP 未加白名单

### 背景

上 CDN 后 sporadic 502，直连源站正常。

### 排查过程

1. 打开 SpeedCE，协议 HTTPS，范围 中国节点
2. 测速结果：CDN 域名 sporadic 红，源站 IP 全绿。
3. 定位根因并修复
4. 复测确认恢复

### 处理结果

源站安全组添加 CDN 回源 IP 段白名单。

### 经验总结

上 CDN 后源站安全策略要同步调整。

## 案例 2：DNS 迁机后新疆持续红

### 背景

迁机完成，团队都说正常，新疆同事坚持打不开。

### 排查过程

1. 打开 SpeedCE，协议 HTTPS，范围 中国节点
2. 测速结果：SpeedCE 地图：新疆持续红，其他省绿且随时间减少。

4. 复测确认恢复

迁机验收不能只看自己省份。

## 案例 3：安全组只放了 22 端口
